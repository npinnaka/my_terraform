# aws_lambda_terraform

Deploy Lambda using Terraform.

## Installation
clone or fork repo.

```bash
git clone git@github.com:npinnaka/tf_dyna_s3_event_lambda.git
```

## Usage

```terraform
terraform init
terraform plan
terraform apply 
```

## output
lambda deployed in aws
