# network

Network data sources/resources 

## Installation
clone or fork repo.

```bash
git clone git@github.com:npinnaka/my_terraform.git
```

## Usage

```terraform
terraform init
terraform plan
terraform apply 
```

## output
vpc id, subnets, security groups
